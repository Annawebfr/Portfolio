Argent Bank

ArgentBank Dans ce projet, j'ai participé au développement front-end d’une application bancaire en utilisant React et Redux, afin de proposer une interface utilisateur moderne, dynamique et réactive. Ma mission principale consistait à intégrer le front-end avec le back-end en mettant en place des appels à l'API REST, assurant ainsi une communication fluide et efficace entre le client et le serveur. J'ai également contribué à la gestion de l'état de l'application avec Redux, permettant une synchronisation optimale des données côté front-end. Objectifs pédagogiques : Mettre en place des routes API pour assurer la communication entre le client et le serveur Implémenter la gestion des états et des données avec Redux Afficher dynamiquement les données issues du back-end via des appels API
